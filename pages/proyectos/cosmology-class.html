Análisis de Modelos Cosmológicos con CLASS + Monte Python

Estudio de parámetros cosmológicos mediante inferencia bayesiana

🌌 Descripción del Proyecto

Actualmente trabajo en la exploración de modelos cosmológicos más allá del modelo ΛCDM estándar. Utilizo CLASS (Cosmic Linear Anisotropy Solving System) para calcular espectros de potencia y funciones de correlación, y Monte Python para realizar inferencia bayesiana mediante cadenas de Markov Monte Carlo (MCMC).

El objetivo es estudiar cómo varían los parámetros cosmológicos (densidad de materia oscura, parámetro de Hubble, índice espectral) al introducir modificaciones en la ecuación de estado de la energía oscura o modelos de materia oscura interactuante.

🔭 Metodología

1. Cálculo del espectro de potencia con CLASS

CLASS resuelve las ecuaciones de perturbación cosmológica para calcular el espectro de potencia de materia P(k) y las funciones de correlación de los modos escalares y tensoriales.

# Parámetros de entrada para CLASS
omega_b = 0.02242
omega_cdm = 0.11933
h = 0.6766
ns = 0.9665
ln10^{10}A_s = 3.047

2. Inferencia bayesiana con Monte Python

Monte Python utiliza algoritmos MCMC para explorar el espacio de parámetros y encontrar las regiones más probables dadas las observaciones (Planck, BAO, SNIa).

📈 Resultados Preliminares

Comparación del espectro de potencia de materia para diferentes valores de Ωcdm. Las cadenas MCMC muestran correlaciones interesantes entre parámetros que estoy analizando actualmente.

  • Ωm: 0.315 ± 0.007
  • H0: 67.4 ± 0.5 km/s/Mpc
  • σ8: 0.811 ± 0.006

Próximos pasos: Implementar modelos con interacción en el sector oscuro y comparar con datos de encuestas de galaxias.
