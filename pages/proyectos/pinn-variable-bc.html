PINN con Condiciones de Frontera Variables

Extensión de tesis: Red Neuronal Físicamente Informada para ecuación de calor 1D

📖 Descripción del Proyecto

Partiendo de mi tesis de licenciatura "Solución de la ecuación de calor unidimensional estacionaria empleando una red neuronal físicamente informada", desarrollé una extensión que permite resolver la misma ecuación pero con condiciones de frontera arbitrarias (variables en el tiempo y en el espacio).

Las Redes Neuronales Físicamente Informadas (PINNs) son una técnica innovadora que incorpora las leyes físicas directamente en la función de pérdida de la red, permitiendo resolver ecuaciones diferenciales parciales sin necesidad de datos de entrenamiento extensivos.

🔬 Metodología

Arquitectura de la Red

La arquitectura PINN incorpora la ecuación de calor como término de pérdida en la función de costo. Implementé una capa de entrada que parametriza las condiciones de frontera, permitiendo que la red generalice a múltiples escenarios sin necesidad de reentrenamiento completo.

# Estructura simplificada de la PINN
def pinn_model(x, t, boundary_conditions):
    # Red neuronal
    u = Dense(50, activation='tanh')(x)
    u = Dense(50, activation='tanh')(u)
    u = Dense(1)(u)
    
    # Ecuación de calor: u_t = α u_xx
    with tf.GradientTape() as tape:
        tape.watch(x)
        u_x = tape.gradient(u, x)
        u_xx = tape.gradient(u_x, x)
    
    # Pérdida física
    physics_loss = tf.reduce_mean((u_t - alpha * u_xx)**2)
    
    # Pérdida por condiciones de frontera
    bc_loss = tf.reduce_mean((u - boundary_conditions)**2)
    
    return physics_loss + bc_loss

Innovación Principal

La clave de esta extensión es que la red aprende a resolver la ecuación para cualquier condición de frontera dentro de un rango definido. Esto se logra alimentando a la red no solo con las coordenadas espaciales y temporales, sino también con los valores de frontera como entradas adicionales.

📊 Resultados

Evolución de la solución para diferentes condiciones de frontera

Los resultados muestran que la red es capaz de predecir soluciones precisas para condiciones de frontera no vistas durante el entrenamiento:

  • Error relativo promedio: < 1% para la mayoría de configuraciones
  • Tiempo de inferencia: 0.01 segundos por predicción (vs minutos para métodos numéricos tradicionales)
  • Generalización: Funciona para condiciones de frontera variables en el tiempo y el espacio
