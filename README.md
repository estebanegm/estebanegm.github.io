# Mi Página Personal

Esta es mi página web personal creada con HTML, CSS y JavaScript. Está diseñada para mostrar mi perfil profesional, proyectos y habilidades.

## 🚀 Características

- Diseño responsive y moderno
- Navegación suave
- Menú hamburguesa para móviles
- Secciones: Inicio, Sobre Mí, Proyectos, Habilidades, Contacto
- Formulario de contacto
- Animaciones al hacer scroll

## 🛠️ Tecnologías Utilizadas

- HTML5
- CSS3 (Flexbox, Grid, Variables CSS)
- JavaScript (Vanilla)
- Font Awesome para iconos

## 📦 Instalación

1. Clona este repositorio
2. Abre `index.html` en tu navegador

## 🚀 Despliegue en GitHub Pages

1. Crea un repositorio en GitHub con el nombre `tuusuario.github.io`
2. Sube todos los archivos a la rama `main`
3. Ve a Settings > Pages y selecciona la rama `main` como fuente
4. Tu sitio estará disponible en `https://tuusuario.github.io`

## 📝 Personalización

1. Reemplaza `Tu Nombre` con tu nombre real
2. Actualiza los textos en las secciones
3. Cambia las imágenes por las tuyas
4. Modifica los colores en las variables CSS
5. Actualiza los enlaces de redes sociales
